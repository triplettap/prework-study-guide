HTML

  • HTML (Hypertext Markup Language) gives the content of a web application structure.
  • HTML is made up of elements - types of content.
  • Common elements used to add content to pages include text, images, headers, and lists.
  • Elements can also work "behind-the-scenes" to divide the content or describe it to browsers and developers.
  • Common "behind-the-scenes" elements are meta-related, content division, and semantic.
  • HTML elements are usually composed of an opening tag and a closing tag represented with angle brackets.
  • Closing tags begin with a forward slash.
  • The head element contains information that helps the browser render the page correctly, and elements included within it are not visible to the user.
  • The body element contains the content that is shown to the webpage user.
  • An HTML attribute defines additional information about an element and is always in the beginning element.
  • Elements that are nested inside other elements are called child elements, and the element that holds all the children is known as the parent element.
  • Semantic HTML elements use the element name to describe its content and function.
  • Section elements are semantic elements that define a section in a document that contains thematic content.

CSS

  • CSS (Cascading Style Sheets) defines how webpage content is presented to users.
  • CSS defines colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Inline CSS styles elements directly in the HTML file.
  • An internal style sheet embeds directly in the HTML file using a style element in the head element.
  • An external style sheet links a seperate file using a link element in the HTML's head element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain the CSS property we want to apply and the value of that property.
  • Class attributes allow us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector.
